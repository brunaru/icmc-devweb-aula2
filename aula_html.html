Anatomia de uma página HTML

Elementos básicos da estrutura de uma página.

  • Páginas HTML5 devem apresentar a seguinte estrutura:
  • !DOCTYPE html: indica que nosso documento trata-se de uma página HTML
  • html: define o início e o fim da minha página
  • head: contém título, encoding, arquivos externos da página (e.g., JS, CSS, fontes, etc)
  • body: contém conteúdo da página.
  • div, section, nav etc: são containers para organizar nosso conteúdo

Links

Ligações que permitem a navegação entre recursos na Web

Links são identificados pela tag "a" e seu atributo "href". Além de URLs, o elemento a suporta:

  • protocolos Web: http://, https://, ftp://
  • #id: # acompanhada do id do elemento é chamada de âncora
  • local de recursos: (./aula_html.html , ./aula_css.html)
  • contatos: mailto:, tel:

A tag "a" pode conter um atributo "target", o qual define onde o recuro irá abrir.

  1. _self: na mesma janela
  2. _blank: em uma nova janela

Imagens

E seus atributos

A tag "img" é utilizada para incluir imagens em páginas HTML. O atributo "src" é utilizado para indicar a localização da imagem. O atributo "alt" é utilizado para indicar o texto alternativo.

Organizando a página em containers semânticos

Boas práticas em HTML5

No dev Web moderno as páginas são organizadas em containers. Exemplo:

  1. header: container dedicado ao cabeçalho da página
  2. nav: container dedicado à navegação da página
  3. footer: container dedicado ao rodapé da página
  4. aside: container dedicado a conteúdos ao lado da página (e.g., propagandas, links externos, feeds, etc)
  5. section: container dedicado a seções da página
  6. article: container dedicado a conteúdos idependentes da página
  7. div: container genérico

Parágrafos

E a diferença entre p e span

Temos usando o elemento p para definir parágrafos. Porém também podemos utilizar o elemento span. No entanto, o span não cria uma nova linha e é geralmente utilizado para formatar trechos dentro de um parágrafo.

O elemento p pula linhas porque sua visualização por padrão é "block", enquanto o span é "inline". Observe que alguns elementos HTML pulam linha automaticamente (e.g., imagens, listas), enquanto outros não. Vamos entender melhor o que "block" e "inline" fazem ao utilizar CSS.

Formulários

E alguns dos seus principais elementos

A tag form define um formulário. Os elementos dentro da tag form são enviados usando GET ou POST. Seus principais atributos são:

  • action: define o gerenciador do formulário. É o endereço de uma página ou serviço.
  • method: define a forma de envio de dados. Pode ser GET ou POST.

Os principais elementos de um form são:

  • input: define vários tipos de entrada no formulário. Exemplos:
    • text: entrada de texto curto.
    • email: entrada do tipo e-mail.
    • radio: entrada de seleção única de valores.
    • color, date, number, file, etc
    • Lista completa de tipos de input no site da w3schools
  • textarea: área de texto. É bem maior que o input de texto. Sua principal utilização é em comentários.
  • select: é uma lista de opções com itens (option). Muitas vezes representada por uma combo box.
  • button: botão de submissão. É uma tag mais atual e mais fácil de customizar que o input do tipo submit.
